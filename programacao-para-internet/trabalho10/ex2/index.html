Exercício 2

Ex2-encadeamento-fetch

A função fetch busca um recurso em "https://ipapi.co/json/".

O primeiro then trata o sucesso da promise e guarda a resposta em formato JSON.

O segundo then é executado utilizando os dados de latitude e longitude obtidos na promise anterior para chamar uma API de dados meteorológicos.

O terceiro then trata a resposta em um JSON.

O quarto then insere alguns dados desse JSON (temperatura e velocidade do vento) no HTML.

O único catch será executado caso qualquer promise seja rejeitada ou lance exceção e apenas mostra a mensagem de erro no console.
