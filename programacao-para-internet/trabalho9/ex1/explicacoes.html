exemplo 1)

Ex1-hello-ajax

index.html

Possui um botão que faz requisição AJAX com XHR para carregar conteúdo de outro arquivo "conteudoAdicional.html". O script ativado pelo botão utiliza o método "open" e faz requisição GET para obter o conteúdo do arquivo. Há então o tratamento para a requisição: em caso de sucesso, "onload" insere o conteúdo; em caso de erro, "onerror" mostra mensagem de erro.

conteudoAdicional.html

Contém conteúdo HTML adicional para ser carregado em outra página.

exemplo 2)

Ex2-busca-cidade

index.html

Possui um campo para inserir um CEP. Ao digitar no campo, é realizada uma requisição XHR GET por um script PHP para busca de cidade utilizando o texto digitado.

busca-cidade.php

O script PHP recebe o input $cep em sua chamada e processa apenas '38400-100' ou '38700-000', retornando o nome das respectivas cidades; no caso de CEP desconhecido, retorna erro.

exemplo 3)

Ex3-recebendo-json1

index.html

Possui campos para CEP, rua, bairro e cidade. É esperado que se digite um CEP e os outros campos sejam preenchidos utilizando o JSON obtido na resposta da requisição XHR GET pelo script "busca-endereco.php".

busca-endereco.php

Recebe um $cep e compara com entradas definidas no código. Se válido, cria um objeto tipo "Endereco" com seus respectivos dados ($rua, $bairro, $cidade) e os retorna codificados no formato JSON.

exemplo 5)

Ex5-enviando-json

index.html

Preenche campos a partir do envio de um JSON pela requisição XHR POST, contendo o CEP digitado, o email 'fulano@mail.com' e a apikey 'abcdefg123456' convertidos para JSON a partir do objeto "objetoJS".

busca-endereco.php

Recebe uma string JSON e a converte em objeto PHP para processar o CEP e retornar outro JSON com os dados do endereço.

exemplo 6)

Ex6-login

index.html

Página de login com os campos "E-mail" e "Senha". Dados são verificados por uma requisição XHR POST enviando os campos do form para o script "checkLogin.php". No caso de falha, é mostrada uma mensagem de erro, e em caso de sucesso, é carregada a página "home.html"

checkLogin.php

Recebe um 'email' e uma 'senha' e os valida. Em caso de sucesso, cria um objeto PHP "RequestResponse" que indica sucesso e a qual página deve ser redirecionado; em caso de falha, cria um "RequestResponse" indicando falha. O script retorna a resposta objeto "RequestResponse" convertido para formato JSON.

home.html

Página aberta em caso de login com sucesso, não tem funções.
